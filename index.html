YelpCamp

Main Page

Lots of Campgrounds!

One detailed campground

Are you want to share your favorite campground to people? If yes, come and add a campground that belongs to you!
YelpCamp is a light-weight web application that can add, edit, comment any campgrounds here. It has been deployed on Heroku. It's main framework lies on Node.js, with MobgoDB to store User data. It uses Express.js to achieve RESTful Routing, Passport.js to maintain data persistence, and Bootstrap to achieve responsive web design.
The highlights:
•   Support people to create their own campgrounds
•   Capable of adding comments to others' campgrounds, edit their own campgrounds
•   Searching campgrounds with particular keywords.
